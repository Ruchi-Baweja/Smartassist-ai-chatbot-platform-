# AI-Powered Customer Support Chatbot Platform

## Project Description
This project is an AI-powered chatbot system that provides customer support using NLP and works across multiple channels like web chat and messaging platforms.

## Features
- AI Chatbot using NLP
- Multi-channel support (Web, WhatsApp, etc.)
- Automated responses
- Ticket management system
- Admin dashboard

## Technologies Used
- React.js
- Node.js
- MongoDB
- Python (NLP)
- Socket.io
